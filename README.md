# Random Task Generator (Генератор случайных задач)

**Автор:** Михайлова Виктория

## Описание

GUI приложение для генерации случайных задач с возможностью добавления новых задач, фильтрацией по типу и сохранением истории в JSON.

### Возможности:
- Генерация случайных задач из предопределенного списка
- 5 типов задач: Учеба, Спорт, Работа, Дом, Хобби
- Добавление пользовательских задач
- История сгенерированных задач
- Фильтрация истории по типу задач
- Сохранение/загрузка истории в JSON
- Валидация ввода при добавлении задач
